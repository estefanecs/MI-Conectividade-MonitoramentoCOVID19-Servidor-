MI-problema1(Paciente)
-Será necessário verificar se a biblioteca do Json está instalada.
-Há a necessidade da instalação do Radmin, para a obtenção do ip público da maquina em que está esse projeto.
-Poderá executar diretamente no netbeans ou acessar a pasta "dist" e executar o arquivo .jar
-Antes de executar qualquer ação, é preciso adicionar o número da porta em que o servidor irá rodar.
-Há a tela para cadastro e remoção do paciente.
-Para acessar a tela do dispositivo de determinado paciente, é preciso informar o nome do mesmo.
